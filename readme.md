Sistema de Registro de Eventos Para H/F Maracay

La idea es crear un sistema que susttuya la plataforma de Eventbrite para poder hacer un control de las
personas que se registran para los eventos de Hackers and Founders

La idea es que en el frontpage aparezca la informacion del evento en una vista que permita ver la descripción
del tema o temas y sea facil para la persona darle registrar y apartar su entrada.

La base de datos de usuarios pudiera enlazarse con la Base de Datos del sitio web de Hackers & Founders y asi
fomentar que la gente se inscriba para los foros y demas actividades.
